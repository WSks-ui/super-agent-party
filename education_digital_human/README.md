# 教育数字人完整系统

基于 Super Agent Party 构建的完整教育数字人系统，专为"普通高等教育组"设计。

## 🎯 系统概述

这不是一个简单的插件，而是一个完整的教育数字人解决方案，充分利用 Super Agent Party 已有的强大功能：

- ✅ **VRM 3D数字人** - 虚拟形象展示
- ✅ **智能体系统** - 角色卡、记忆系统
- ✅ **知识库** - FAISS向量数据库 + BM25检索
- ✅ **工具调用** - 文献检索、数据分析、代码执行
- ✅ **多模态** - 语音识别、语音合成
- ✅ **长期记忆** - 跟踪学习进度

## 🚀 快速开始

### 第一步：配置模型提供商

1. 打开 Super Agent Party
2. 进入【设置】->【模型提供商】
3. 添加你的 LLM 提供商（OpenAI、Ollama、Dify等）
4. 测试连接确保可用

### 第二步：导入角色卡

1. 进入【角色】界面
2. 点击【导入角色卡】
3. 选择 `角色卡/虚拟科研导师.json`
4. 角色卡会自动加载，包含完整的系统提示词

### 第三步：构建知识库

1. 进入【知识库】界面
2. 点击【创建知识库】
3. 命名为"教育研究方法"
4. 上传 `知识库/教育研究方法.md` 文件
5. 点击【处理知识库】，等待向量索引构建完成

### 第四步：配置工具链

1. 进入【设置】界面
2. 参考 `工具链/教育研究工具配置.json` 启用以下工具：
   - ✅ 网络搜索（用于文献检索）
   - ✅ Wikipedia（获取基础概念）
   - ✅ arXiv（检索学术论文）
   - ✅ 代码解释器（数据分析）
   - ✅ 知识库（使用教育研究方法库）
   - ✅ 时间工具（研究规划）
   - ✅ 文件读取（读取研究文档）

### 第五步：配置VRM数字人（可选）

1. 进入【VRM】界面
2. 上传你的 VRM 模型文件（.vrm格式）
3. 配置语音（推荐使用EdgeTTS的YunxiNeural男声）
4. 启用表情和动作

### 第六步：配置记忆系统

1. 进入【设置】->【记忆】
2. 启用记忆功能
3. 设置记忆限制为50轮
4. 设置用户名为"同学"

## 🎓 四大核心功能模块

### 1. 虚拟科研导师 👨‍🏫

**特点：**
- 引导式提问，而非直接给出答案
- 方法论指导，帮助设计严谨的研究方案
- 协作者定位，而非信息检索工具

**使用场景：**
```
学生：张教授，我想做一个关于AI在教育中应用的研究
张教授：很好的方向！让我们一起探讨几个问题：
1. 你的具体研究问题是什么？
2. 你打算用什么研究方法？
3. 你希望解决什么教育痛点？

你觉得呢？
```

### 2. 文献协作检索 📚

**利用工具：**
- Web搜索：查找最新研究
- arXiv：检索学术论文
- 知识库：获取研究方法指导

**协作流程：**
1. 学生提出研究主题
2. 数字人协助设计检索策略
3. 共同筛选相关文献
4. 引导学生批判性阅读

### 3. 实验设计协作 🔬

**功能：**
- 帮助确定研究问题和假设
- 指导选择合适的研究方法
- 协助设计实验方案
- 讨论伦理考量

**利用工具：**
- 知识库：研究方法参考
- 代码解释器：样本量计算、统计分析
- 时间工具：研究时间线规划

### 4. 学习进度跟踪 📊

**利用记忆系统：**
- 自动记录对话历史
- 跟踪研究进展
- 记录学习里程碑
- 回顾成长历程

## 💡 使用技巧

### 1. 如何开始一次科研协作

```
你：张教授，我想研究一下机器学习在个性化学习中的应用
张教授：非常好的研究方向！让我们从几个问题开始：

1. 你关注的具体教育场景是什么？（例如：K12、高等教育、职业培训？）
2. 你希望解决个性化学习中的什么痛点问题？
3. 你有相关的数据来源吗？

你觉得呢？
```

### 2. 如何进行文献检索

```
你：能帮我找一些关于AI教育应用的最新文献吗？
张教授：当然！在开始检索之前，我们先明确一下：

1. 你想关注哪些具体的AI技术？（机器学习、深度学习、NLP？）
2. 你关注的教育应用场景是什么？
3. 有特定的时间范围要求吗？

明确这些后，我可以帮你设计更精准的检索策略。你觉得呢？
```

### 3. 如何进行实验设计

```
你：我想设计一个实验来验证我的假设
张教授：很好！让我们一步步来：

首先，你的研究假设是什么？请用"如果...那么..."的句式表述。

然后我们来讨论：
- 自变量是什么？
- 因变量是什么？
- 你打算如何测量？
- 需要控制哪些变量？

你先试着表述一下你的假设，我们一起来完善。
```

## 📝 注意事项

### 学术诚信

- 始终明确标注AI生成的内容
- 鼓励学生批判性评估AI输出
- 强调人类研究者的主体地位

### AI局限性

- 明确告知学生AI可能的错误
- 鼓励多方验证重要信息
- 强调AI是辅助工具，不能代替人类思考

### 数据安全

- 研究数据存储在本地
- 不上传到任何服务器
- 注意保护研究参与者隐私
